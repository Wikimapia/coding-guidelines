JavaScript requires semicolons to delimit statements.

Never rely on automatic semicolon insertion. **It will burn you.**

- Use semicolons

    Yes

    ```
    var semicolons = 'YES';
    ```

    No

    ```
    var semicolons = 'WATMEWORRY'
    ```

Semicolons are **not** required for:

    * for
    * function
    * if
    * switch
    * try
    * while
